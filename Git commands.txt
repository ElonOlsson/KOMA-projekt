Git Commands

cd Documents - tar dig till mappen documents
cd .. - tar dig tillbaka ett steg 
ls - listar allting i en mapp
git status - visar statusen för git
git add “filnamn” - lägger till en fil för att sedan commit
git add . - lägger till alla filer för att sedan commit
git commit -m “meddelande” - commit dina ändringar med ett meddelande
git pull - uppdaterar din lokala mapp med ändringar som andra gjort
git push - skickar ut dina lokala ändringar så att de andra kan pull
